# Brief: UI And Feedback References

## Goal

Create reference visuals for the first demo HUD and feedback presets.

## Runtime Targets

- HUD reference for bottom target list.
- Timer, score, combo, hint, and magnifier visual references.
- Feedback references for `find_success`, `pop_success`, `catch_success`, `hint_reveal`, and `wrong_tap`.

## Prompt Draft

Design a cozy mobile hidden-object game HUD and feedback visual reference. The HUD should include a compact top timer and score area, a bottom target list with item icons and counts, and small round tool buttons for hint and magnifier. The style should match a warm hand-drawn casual game, with clear touch targets, restrained decoration, and readable contrast. Also create visual references for success ring with sparkle, balloon pop puff, cute catch stamp, hint pulse highlight, and wrong tap shake ripple. Avoid marketing hero layouts, heavy gradients, text-heavy instruction panels, watermarks, and complex decorative frames.

## Acceptance Notes

- HUD leaves the map as the main focus.
- Controls are readable on mobile.
- Feedback states are clear without covering too much of the target area.
- Visuals can be implemented in Cocos with sprites, particles, tweens, and simple audio hooks.
