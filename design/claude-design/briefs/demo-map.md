# Brief: Demo Cozy Town Map

## Goal

Create the first large hidden-object map background for the demo.

## Runtime Target

- Runtime path: `art/maps/demo_cozy_town_placeholder`
- Suggested size: `2400x1600`
- Format after normalization: PNG or WebP

## Prompt Draft

Create a cozy hand-drawn hidden-object game map called Demo Cozy Town. The scene should feel warm, playful, and calm, with soft lines and clear object silhouettes. Use a small town park and street market composition with many readable hiding spots: stalls, benches, planters, windows, signs, baskets, trees, blankets, and small decorative props. The image should be bright and friendly, suitable for mobile gameplay, with enough detail for hidden objects but no clutter that makes interaction unreadable. Avoid text labels, watermarks, UI overlays, harsh shadows, aggressive shapes, and dark mood.

## Composition Notes

- Leave useful detail across the whole image, not only the center.
- Keep important hiding regions away from extreme edges.
- Maintain clear depth layers but avoid heavy blur.
- Make object scale consistent enough that target points can be tapped.

## Acceptance Notes

- Works as a full-map background.
- Has at least 30 plausible hiding positions.
- Remains readable when zoomed out to show most of the map.
- Does not contain baked UI.
