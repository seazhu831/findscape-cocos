# Brief: Demo Target Set

## Goal

Create target sprites and target-list icons for the first demo object categories.

- `art/icons/tool_magnifier`

## Prompt Draft

Create a cohesive set of cozy hand-drawn mobile game objects on transparent backgrounds: pineapple, balloon, cute sneaky thief character, puppy, gem, hint tool icon, and magnifier tool icon. Use soft rounded shapes, clean silhouettes, warm friendly colors, and consistent line weight. Each object should be readable at small mobile UI sizes. Provide both gameplay target versions and simplified square icon versions where useful. Avoid text, labels, watermarks, photorealism, harsh outlines, and overly detailed texture.

## Output Notes

- Target sprite source: `512x512` transparent PNG.
- Icon source: `256x256` transparent PNG.
- Keep each object centered with moderate padding.
- Preserve a consistent visual family.

## Acceptance Notes

- Each target can be recognized at list-icon size.
- Target sprites can be placed into the map without feeling stylistically foreign.
- Thief remains cute and non-threatening.
