# Assets

Runtime assets for the Cocos Creator client live here.

Do not hand-create Cocos `.meta` files. Let the chosen Cocos Creator version generate and maintain them after the project is opened in the editor.

Suggested top-level areas:

- `scripts/`: TypeScript source modules.
- `resources/config/`: data-driven gameplay configuration.
- `resources/art/`: normalized runtime art exported from Claude Design or other production sources.
- `resources/audio/`: sound effects and music hooks.
- `scenes/`: Cocos scenes after the editor project is initialized.
- `prefabs/`: reusable Cocos prefabs after UI and gameplay objects are built.
